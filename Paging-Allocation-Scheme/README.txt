MemMan is a program that demonstrates first fit and best fit memory allocation schemes.
To execute MemMan:

1. Place MemMan.c in your desired directory.
2. In the terminal, navigate to that directory.
3. Type the following command to compile the program: gcc MemMan.c -o MemMan
4. Type the following command to run the program: ./MemMan
5. Follow the prompts in the program.
